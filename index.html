For now you can try to open:
  • Learn Vue Fundamentals - Shopping List
    Created with Vue2 to learn the Vue fundamentals, especially on two ways data binding.
    Github Repository: Learn Vue Fundamentals
  • Learn Vue3 Basic - Task Reminder
    Created with Vue3 to learn the Vue3 basic and consumes API from backend endpoint that included in the project using json-server, especially on two ways data binding & fetching API.
    Because it has backend, the data is persistent. So please be wise 👌, thanks.
    I also add Github Action (Workflows) to automatically deploy compiled static files to Github Pages.
    Github Repository: Learn Vue3 Basic
  • Typescript Monorepo - Chat App
    Learning Monorepo by using Typescript with Express & React to create simple chat app. This project also implement tRPC and WebSockets for real-time feature. Also there is branch called basic-chat-app that using tRPC with React Query to pooling data as an alternative for WebSockets.
    Github Repository: Typescript Monorepo
  • GraphQL Basic - Chat App
    Learning GraphQL with Apollo Server as backend and local database for simplification using lowdb (local json file adapter). Have type User and enum of Gender. Query: get all users, get user by id, and follow (add following and followers data to user in action).
    Github Repository: GraphQL Basic
  • Anagram Words - NPM Package
    Repository of NPM Package I published with simple function, to return anagram words if existed from the given string/text, check if two strings are anagram word. There are 2 functions: findPattern & isAnagram that described well in the readme section.
    Github Repository: Anagram Words
  • ReactPHP Project - Websocket Chats
    Example ReactPHP Project : Websocket Chats. With the help of Thruway as PHP Router Library for Autobahn (use autobahn-browser as a client) Real-Time Application Messaging.
    Github Repository: ReactPHP Thruway - Websocket Chats
  • Image Augmentation for ML
    Image augmentation for machine learning experiments. Based on aleju/imgaug and implemetation of asetkn. It basically Jupyter Notebook with ready to use script which supposed to create augmentation of the given images. The images need to be labelled or annotated. The output is augmented version of the images and its annotation files in .xml PASCAL POC format.
    Github Repository: Image Augmentation for ML
